JavaScript

JavaScript, often abbreviated as JS, is a programming language and core technology of the World Wide Web, alongside HTML and CSS. In JavaScript, the String and Math objects provide a variety of functions for working with strings and performing mathematical operations, respectively. Let's explore the functions in each of these objects in detail.

String Object

The String object is used to manipulate sequences of characters in JavaScript. Here are some commonly used functions:
  1. charAt(index)
    2. - Returns the character at the specified index in a string. 
                            let str = "Hello";
                        console.log(str.charAt(0)); // Outputs: "H"
  2. concat(str1, str2, ...)
    3. - Combines two or more strings and returns a new string. 
                            let str1 = "Hello";
                        let str2 = " World";
                        let result = str1.concat(str2);
                        console.log(result); // Outputs: "Hello World"
  3. indexOf(substring)
    4. - Returns the index of the first occurrence of a specified substring. 
                            let str = "Hello World";
                        console.log(str.indexOf("World")); // Outputs: 6
  4. substring(startIndex, endIndex)
    5. - Returns a substring based on the specified start and end indices. 
                            let str = "Hello World";
                        console.log(str.substring(6, 11)); // Outputs: "World"
  5. toUpperCase() / toLowerCase()
    6. - Converts a string to uppercase or lowercase. 
                            let str = "Hello";
                        console.log(str.toUpperCase()); // Outputs: "HELLO"
                        console.log(str.toLowerCase()); // Outputs: "hello"
  6. split(separator)
    7. - Splits a string into an array of substrings based on a specified separator. 
                            let str = "apple,orange,banana";
                        let fruits = str.split(",");
                        console.log(fruits); // Outputs: ["apple", "orange", "banana"]

Math Object

The Math object provides mathematical functions and constants in JavaScript. Here are some commonly used functions:
  1. abs(x)
    2. - Returns the absolute value of a number. 
                            console.log(Math.abs(-5)); // Outputs: 5
  2. ceil(x) / floor(x)
    3. - Rounds a number up (ceil) or down (floor) to the nearest integer. 
                            console.log(Math.ceil(4.3)); // Outputs: 5
                        console.log(Math.floor(4.9)); // Outputs: 4
  3. max(x, y, ...) / min(x, y, ...)
    4. - Returns the maximum or minimum value from a set of numbers. 
                            console.log(Math.max(10, 5, 8)); // Outputs: 10
                        console.log(Math.min(10, 5, 8)); // Outputs: 5
  4. random()
    5. - Returns a random floating-point number between 0 (inclusive) and 1 (exclusive). 
                            console.log(Math.random()); // Outputs: a random number between 0 and 1
  5. sqrt(x)
    6. - Returns the square root of a number. 
                            console.log(Math.sqrt(25)); // Outputs: 5
  6. pow(x, y)
    7. - Returns the result of raising x to the power of y. 
                            console.log(Math.pow(2, 3)); // Outputs: 8
